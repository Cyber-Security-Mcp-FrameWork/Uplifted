把最强的 LLM（GPT-4, Claude, Gemini）和 600+ 预构建的黑客工具结合，让 AI 成为你的安全研究员、渗透测试员、威胁猎人。自动化执行那些重复无聊的活儿，让你专注于真正有趣的挑战。

```python

# 让 Agent 自己决定如何扫描
response = requests.post(f'http://localhost:7541/api/v1/agents/{agent_id}/run', json={
```

### Example 3: 实时威胁监控

```python
import websocket
import json

# 建立 WebSocket 连接进行实时监控
ws = websocket.create_connection("ws://localhost:7541/ws/monitor")

# 配置监控规则
ws.send(json.dumps({
    "action": "start_monitoring",
    "config": {
        "sources": ["/var/log/auth.log", "/var/log/syslog"],
        "patterns": ["failed login", "unauthorized access", "privilege escalation"],
        "ai_analysis": True,  # 启用 AI 分析
        "auto_response": True  # 自动响应
    }
}))

# 接收实时告警
while True:
    alert = json.loads(ws.recv())
    print(f"[!] ALERT: {alert['message']}")
    print(f"[*] Severity: {alert['severity']}")
    print(f"[*] AI Analysis: {alert['ai_insight']}")
    print(f"[*] Recommended Action: {alert['recommendation']}")
```

### Example 4: 自定义工具集成

```python
from server.uplifted.extensions import PluginManager, MCPPluginBridge

# 把你自己的脚本变成 MCP 工具

    def scan(self, target):
        # Your custom scanning logic
        return {"vulns": [...]}

# 注册到 Uplifted

manager.register_plugin("my_scanner", MyCustomScanner())
bridge.register_plugin(manager.get_plugin("my_scanner"))

# 现在 AI Agent 可以使用你的工具了
```
